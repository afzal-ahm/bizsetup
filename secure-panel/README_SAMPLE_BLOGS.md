## How to Use

### Option 1: Run the SQL File (Recommended)
1. Open your database management tool (phpMyAdmin, MySQL Workbench, etc.)
2. Run the `sample_company_gst_blogs.sql` file directly
3. The file will automatically:
   - Create the `ca_website` database if it doesn't exist
   - Create all required tables (`blog`, `blog_categories`, `blog_tags`, `blog_tag_relationships`)
   - Insert 6 CA service categories
   - Insert 6 relevant tags
   - Insert all 5 sample blog posts
   - Set up proper tag relationships

### Option 2: Manual Import
1. First run the category and tag updates
2. Then run the blog post insertions
3. Finally run the tag relationships

### Option 3: Use the Blog Management Interface
1. Go to `blog.php` in your secure panel
2. Use the form to manually add blog posts
3. Copy content from the sample posts as needed
